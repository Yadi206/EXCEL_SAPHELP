# EXCEL_SAPHELP

#### 介绍
EXCEL VSTO开发的辅助工具,可以用来链接SAP调用RFC
已经实现功能:
1.离线读取SAP常用表
2.离线读取表结构
3.在线读取表结构

 
#### 安装教程
https://www.yuque.com/tiger-fb9jj/zpgg4d/ftklu0ougixp8rsa?singleDoc
1.先安装.NET 4.6.2 ,这个是 调用RFC所必须的运行时;
  要等6.2安装完成，再安装工具包
2.再安装 VOST工具包,是运行excel扩展工具的运行时;
安装完重启计算机
3.再安装Excel_saphelp.
4.打开excel,会触发工具安装.
如果没启动,需要重启计算机
